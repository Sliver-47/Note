# 个人记事本 - 添加到手机主屏幕教程

## 方法一：PWA（推荐）

### 步骤

1. **部署到网络服务器**

   首先需要将这些文件部署到一个HTTPS网站上。你可以使用：
   - GitHub Pages（免费）
   - Netlify（免费）
   - Vercel（免费）
   - 或你自己的服务器

2. **准备图标文件**

   我已经为你创建了SVG格式的图标 `icon.svg`，现代浏览器都支持SVG图标。如果某些平台不支持，你可以：
   - 直接使用在线工具转换SVG到PNG：https://convertio.co/zh/svg-png/
   - 或者使用在线图标生成器生成PNG图标

3. **在手机上访问网站**

   使用手机浏览器访问部署好的网站

4. **添加到主屏幕**

   **Android/Chrome:**
   - 点击浏览器菜单（三个点）
   - 选择"添加到主屏幕"或"安装应用"

   **iPhone/Safari:**
   - 点击分享按钮
   - 选择"添加到主屏幕"

## 方法二：使用在线打包工具（快速）

如果你不想自己部署，可以使用以下在线工具：

1. **Web2App** (https://web2appmaker.com/)
2. **Web to App** (https://gonative.io/)
3. **Appy Pie** (https://www.appypie.com/app-builder)

这些工具可以将你的HTML文件打包成APK（Android）或IPA（iOS）文件，你可以安装到手机上。

## 方法三：使用本地服务器测试

如果你想先在本地测试：

1. 在当前目录运行一个简单的HTTP服务器：

```bash
# 使用Python
python -m http.server 8000

# 或者使用Node.js（如果已安装）
npx http-server
```

2. 确保手机和电脑在同一WiFi网络

3. 在手机浏览器访问 `http://你电脑的IP:8000/记事本.html`

## 文件说明

- `记事本.html` - 主应用文件
- `manifest.json` - PWA应用清单
- `icon.svg` - 应用图标源文件
- `使用说明.md` - 本文件

## 注意事项

- PWA需要HTTPS才能正常工作（本地localhost除外）
- 数据保存在浏览器localStorage中，清除浏览器数据会丢失笔记
- 建议定期使用应用内的导出功能备份笔记
